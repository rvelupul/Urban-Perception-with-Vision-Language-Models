# Urban-Perception-with-Vision-Language-Models

Please note: This repository contains all code developed throughout the semester for the Urban Perception with Vision-Language Models project, progressing from prompting strategies and fine-tuning experiments to a simplified UrbanVLM dual-modality pipeline — the most recent implementation, Urban_VLM_Joint_Classification_and_Regression_Training_Safety_Attribute.ipynb, extends the pipeline by implementing the combined classification and regression objective function from the UrbanVLM paper (IJCNN 2025) and was presented in the final meeting.

Chain_of_Thought_Prompting_and_Independent_Scoring_Evaluation_using_LLaVA_1_5_7b.ipynb

This notebook explores three prompting strategies for urban perception evaluation using LLaVA-1.5-7b on the Place Pulse 2.0 dataset, without any fine-tuning. The dataset is preprocessed by removing tie outcomes and applying a majority ratio filter of 0.7 to retain only pairs with clear human consensus. Three evaluation strategies are implemented: (1) Direct zero-shot prompting, where both images are shown side by side and LLaVA is asked to select the winner, with positional bias mitigated by evaluating each pair in both orders; (2) Chain-of-thought (CoT) prompting, where LLaVA is guided through a structured four-step reasoning process — describing each image, comparing them, and then deciding — using attribute-specific visual cues for all six perception categories; and (3) Independent scoring evaluation, where LLaVA assigns a 0-100 perception score to each image individually rather than comparing two images directly, with the predicted winner determined by comparing the two scores. Results showed that independent scoring consistently outperformed direct pairwise comparison and chain-of-thought prompting, establishing the motivation for moving toward a training-based Q-score approach in subsequent phases.

Direct_Preference_Optimization.ipynb

This notebook implements multimodal Direct Preference Optimization (mDPO) to fine-tune LLaVA-NeXT 7B (llava-hf/llava-v1.6-mistral-7b-hf) on the Place Pulse 2.0 dataset for urban perception comparison tasks. Standard DPO on vision-language models suffers from a well-documented problem where the model ignores the image entirely and relies on language shortcuts — a problem confirmed in initial experiments through a 100% position-swap tie rate. To address this, mDPO (EMNLP 2024) is implemented with two key additions: an image preference loss that contrasts chosen images against degraded versions (blur and color jitter) to force the model to actually look at the images, and a reward anchor that prevents the likelihood of chosen responses from decreasing during training. The total training loss combines the standard DPO loss, the image preference loss, and the anchor loss. QLoRA is used for parameter-efficient fine-tuning on an A100 GPU. Despite these fixes, the model continued to exhibit yes-bias and positional bias — collapsing to a tie rate of 1.0 regardless of image content — confirming that pairwise image comparison through fine-tuning is a fundamental limitation of vision-language models rather than a training configuration issue.

SFT_Supervised_Fine_Tuning.ipynb

This notebook implements Supervised Fine-Tuning (SFT) of LLaVA-NeXT 7B (llava-hf/llava-v1.6-mistral-7b-hf) using QLoRA and PyTorch Lightning on the Place Pulse 2.0 dataset for urban perception pairwise comparison. The model is trained to directly predict which of two street view images — Image A or Image B — is perceived as safer, more beautiful, wealthier, livelier, more boring, or more depressing. To prevent positional bias during training, the winner is placed randomly on the left or right side of each pair. Only high-confidence pairs with a majority ratio of 0.7 or above are used for training. Chain-of-thought evaluation with position swapping is used to assess performance against the baseline CoT results from the prompting notebook. Despite these design choices, the model learned a positional shortcut — consistently predicting the same position regardless of image content — resulting in a tie rate of 1.0 and confirming that supervised fine-tuning for pairwise image comparison is fundamentally limited by positional bias in vision-language models. This finding, alongside the DPO experiments, motivated the pivot to a Q-score regression approach in subsequent phases.

Urban_VLM_Joint_Classification_and_Regression_Training_Safety_Attribute.ipynb

This notebook implements the combined objective function from Stage 4 of the UrbanVLM paper (IJCNN 2025), extending the simplified dual-modality pipeline to train both a classification and regression head simultaneously on the safety attribute. Q-scores are computed from raw pairwise votes as a win-loss ratio scaled to 0-10, with binary labels derived using the median threshold to guarantee a balanced 50/50 class split. LLaVA-1.5-7b generates safety-specific captions for each image using the Appendix D prompt structure from the paper. CLIP ViT-B/32 then encodes both the image (512-dim) and caption (512-dim), which are concatenated into a 1024-dim combined representation. This is passed to the JointUrbanVLMHead — a shared backbone (Linear 1024→256 + ReLU + Dropout 0.3) that branches into a classification head (Linear 256→2) trained with CrossEntropyLoss and a regression head (Linear 256→1) trained with MSELoss. The total loss is L_total = λ_class × CrossEntropyLoss + λ_reg × MSELoss, with both lambda values set to 1.0. Training uses AdamW with cosine annealing over 20 epochs, saving the best checkpoint per lowest MAE. Results are reported for both image-only (512-dim) and image+caption (1024-dim) configurations, with the image+caption model achieving 78.4% classification accuracy and a recall of 0.942 on the safety attribute.

Urban_VLM_Pipeline_using_Dual_Modality_LLaVA_and_CLIP.ipynb

This notebook implements a simplified version of the UrbanVLM pipeline (IJCNN 2025) on the Place Pulse 2.0 dataset, evaluated across all six perception attributes — safety, beauty, wealth, liveliness, boredom, and depression. Pairwise human votes are converted into continuous Q-scores (0-10) per image per attribute using a win-loss ratio, with a minimum threshold of 10 comparisons per image to ensure reliability. LLaVA-1.5-7b generates attribute-specific captions for each street image using the prompt structure from Appendix D of the paper. CLIP ViT-B/32 then encodes both the image (512-dim) and the caption (512-dim) separately, which are concatenated into a 1024-dim dual-modality feature vector. A Random Forest regressor and a lightweight MLP neural network head are both trained on these features to predict the continuous Q-score, with the neural network producing the final results. An explicit two-stage ablation is included for every attribute — comparing image-only (512-dim) against image+caption (1024-dim) features — to directly measure the contribution of adding captions. The dual-modality approach improved performance on 4 out of 6 attributes, with the largest gains on Safer (+11.6%) and More Depressing (+9.6%), confirming the paper's core finding that combining images with text descriptions outperforms image-only approaches.
